## ✅ 숙제 제출 방식 
1. 기본 작업환경은 본인 이름의 브랜치에서 작업합니다.
```
ex) > jaaeun
```
3. 주차별 공부한 내용을 정리 ( md,pdf,블로그 링크 등 ) 후 주차별 폴더에 자기 이름의 폴더를 생성한 후 정리한 내용을 커밋합니다.
```
ex) > week1/jaeeun/1~2주차_정리.py
```
4. 발표 시간 전까지 커밋한 내용을 푸쉬후 PR 합니다.
5. 방장은 발표가 끝난 후 머지합니다.
